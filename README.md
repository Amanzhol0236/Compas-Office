# Compas Office v1

Первый демонстрационный прототип внутреннего приложения для сотрудников Compas Telecom.

## Что есть в первой версии
- Вход по логину и паролю
- Общий корпоративный чат
- Список сотрудников
- Онлайн-статус
- Логотип на каждой странице
- Блоки будущих модулей: уведомления, документы, заявки

## Как запустить
1. Установите Node.js с официального сайта nodejs.org
2. Распакуйте архив
3. Откройте папку проекта в терминале
4. Выполните команды:

```bash
npm install
npm run dev
```

5. Откройте адрес, который покажет терминал, обычно:

```text
http://localhost:5173
```

Для демо можно ввести любой логин и пароль.
